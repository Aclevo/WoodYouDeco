# WoodYouDeco

![WoodYouDeco Image](https://media.forgecdn.net/attachments/1524/60/untitled-png.png)

Enhance your Minecraft World with new enhanced wood types and materials! Upgrade your wood with these new materials not seen in Vanilla Minecraft! Not enough for you? Fork the code and add your style to it!
